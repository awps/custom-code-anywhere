== Description ==
An easy way to add tracking codes or inline javascript to your site without modifying the current theme.

You know, adding code from from services such as Google Analytics requires to edit the theme source code. Doing this is not a good practice. There are many reasons why not. This plugin solves this problem by creating an admin page where you can add the tracking codes or even any javascript.
